Procedurally Generated Toy Town

Procedurally generated diorama, the setting is a "Toy Town", with the design and scale of the city drawing influence from popular MMORPG Runescape. The character models interact with key environmental objects in a basic manner, whilst minor environmental objects such as stones and grass are spawned within a grid.

Student Budgeting Application

The aim of this project was to create a budgeting application catering to the needs of students. Budgeting is generally seen as tedious by this demographic so I attempted to add an element of gamification to determine whether it would encourage engagement.
